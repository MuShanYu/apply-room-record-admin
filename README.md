# 房间预约与流动统计后台管理客户端

## 前言

本项目基于框架Vue-Element-Admin开发，拥有通用动态路由权限，按钮级权限处理，丰富的功能，美观的报表。您可以将其作为项目开发框架，可以大大缩减您的开发时间，或者您也可以抽取其中的部分功能进行二次开发。如果本项目对您有作用，您可以点一个star，感谢！
